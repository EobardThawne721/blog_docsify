*  docsify 锁定某个特定版本，可以在 URL 中的符号后面指定完整版本
```html
<link
  rel="stylesheet"
  href="//cdn.jsdelivr.net/npm/docsify@4.11.4/themes/vue.css"
/>
<script src="//cdn.jsdelivr.net/npm/docsify@4.11.4"></script>
```

```HTML
  <nav>

  </nav>
```
